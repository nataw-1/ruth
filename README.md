# Axum Scholars (Plain HTML/CSS/JS)

Multi-page website built with **plain HTML, CSS, and JavaScript**.

## Pages
- `index.html` (Home)
- `about.html` (About Us)
- `founder.html` (Founder)
- `course1.html` (Course I)
- `course2.html` (Course II & III)
- `contact.html` (Contact)

## How to run
- **Option A (simplest)**: double-click `index.html` to open in your browser.
- **Option B (recommended)**: use a local server (example: VS Code “Live Server”) and open `index.html`.

## Shared assets
- `assets/css/styles.css`
- `assets/js/main.js`
